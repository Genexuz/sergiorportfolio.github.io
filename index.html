Soul of steam is my first game ever made. We were a group of 3 people, myself being one of the gameplay programmers and game designers. It was fully written in Lua and made on LÖVE2D for Windows. It's a 2D action platformer set in a steampunk, retro-futuristic world where you have to defeat souless robots in order to make the world a better place.

Star Grazer is a multiplayer browser game written in Javascript and MySQL database. We were a group of 3 people, myself being one of the gameplay programmers, database programmer and game designer. It's a 2D top-down base builder resource management game where you advance your technologies and go on missions in order to explore further into the universe.
